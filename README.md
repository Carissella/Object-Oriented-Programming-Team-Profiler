  ## Description 

  This team profile generator app is a command-line application that will generate a website that will display the team members of a software engineering team. The app will prompt the user to input information about each team member. The information regarding each manager, engineer, and intern will generate an HTML file a formatted page displaying the team members' information. The information that was input will display names, roles, ID numbers, email addresses, and certain information that will depend on the role being added to the team..

  ## Usage 

  To use the OOP Team Profile Generator, please follow these steps:

1. Please install Node.js on your computer.
2.  Please download or clone this repository to your local computer.
3. Then open the integrated terminal at the root directory of the project or in the index.js file
4. Run npm i inquirer@8.2.4. This will download the dependency needed to run the application.
5. Then run "npm start" in your terminal to begin the prpcess and be prompted with questions.
6. Please answer each of the prompts about each team member. There is no limit to the amount of members the user can add to a team.
7. When you're finished answering the prompts or no longer wish to add more members, please select the "Finish building team" option to generate the new HTML file. It will be saved in the dist folder un the name index.html.
8. Once the HTML file has been generated, please open it in a web browser to view the newly generated team's information.
